# Kylle Ian D. Acibron - Full Stack Developer Portfolio

A modern, responsive, and accessible portfolio website showcasing projects, skills, and professional experience.

## 🚀 Features

### Core Features
- **Responsive Design**: Optimized for all devices (mobile, tablet, desktop)
- **Dark/Light Theme**: Toggle between themes with system preference detection
- **Smooth Animations**: Intersection Observer-based scroll animations
- **Project Filtering**: Dynamic search and category filtering
- **Contact Form**: Validated form with success/error handling
- **Skill Progress Bars**: Animated skill visualization

### Accessibility & Performance
- **WCAG 2.1 AA Compliant**: Semantic HTML, ARIA labels, keyboard navigation
- **SEO Optimized**: Meta tags, structured data, semantic markup
- **Performance Optimized**: Lazy loading, debounced events, efficient DOM queries
- **Reduced Motion Support**: Respects user motion preferences
- **Screen Reader Friendly**: Proper heading structure and descriptive labels

### Enhanced Features
- **Typing Animation**: Dynamic role display with realistic typing effect
- **Smooth Scrolling**: Navigation with offset for fixed header
- **Mobile Menu**: Hamburger menu with accessibility features
- **Micro-interactions**: Hover effects, transitions, and visual feedback
- **Easter Egg**: Konami code activation (↑↑↓↓←→←→BA)
- **Loading States**: Smooth page load animations

## 🛠️ Technologies Used

### Frontend
- **HTML5**: Semantic markup with accessibility features
- **CSS3**: Custom properties, Grid, Flexbox, animations
- **JavaScript ES6+**: Modern features, modular architecture
- **Font Awesome**: Icon library
- **Google Fonts**: Inter font family

### Development Practices
- **Mobile-First Design**: Progressive enhancement
- **Component-Based CSS**: Organized, maintainable styles
- **Performance Budget**: Optimized assets and code
- **Accessibility Testing**: Screen reader and keyboard navigation
- **Cross-Browser Compatibility**: Modern browser support
